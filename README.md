# Diabetes Readmission Prediction

This repository contains a simple Streamlit app and model artifacts to predict diabetes readmission.

Quick start

- Build and run with Docker Compose:

```bash
docker-compose up --build
```

- Run locally in a virtual environment:

```bash
python -m venv .venv
.
# Windows PowerShell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```

Actuellement, personne ne le sait avec certitude.  Mon projet consiste à créer un système qui répond à cette question automatiquement.
