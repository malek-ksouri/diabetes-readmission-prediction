# diabetes-readmission-prediction
Imagine un hôpital.  Un patient diabétique sort aujourd'hui de l'hôpital.  Le médecin aimerait savoir :  "Est-ce que ce patient risque de revenir à l'hôpital dans les 30 prochains jours ?"  Actuellement, personne ne le sait avec certitude.  Mon projet consiste à créer un système qui répond à cette question automatiquement.
